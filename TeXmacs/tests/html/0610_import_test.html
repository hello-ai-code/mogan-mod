分析与讨论

  1. 误差分析
    1. 第1，2号小球的流体静力称衡法测量结果均比直接测量法大，可能有以下误差来源：
      1. 螺旋测微器存在误差，本人所在实验台的螺旋测微器由于上一组实验者的不当操作遭到了损坏，测量结果可能存在一定偏差
      2. 实验二中使用的纯水密度为室温（20℃）下数据，实际夏季实验室室温高于该值，导致测量结果偏大
      3. 将小球放入水中，小球表面沾有气泡，导致实际排水体积大于小球体积，最终测量结果偏大
    2. 3号小球的流体静力称衡法测量结果比直接测量法小...
    3. 其他误差来源：
      1. 样品放入或取出样品笼时的微小晃动/碰撞/影响实验结果
      2. 样品浸入后，吊丝或样品表面与液面之间的表面张力会产生附加力，影响测量
      3. 水中有杂质、气泡、油污等，改变实际密度
      4. 人为读数、记录、计算过程中的误差
  2. 可能的改进方向
    1. 样品放入水中时，用镊子轻轻敲打样品以排出气泡。 放入前应确保样品表面干净无杂质，减少气泡吸附
    2. 修正液体密度/使用恒温的水浴控制水温
    3. 使用多种液体测量，修正一部分系统误差
  3. 收获与感想
    1. 理解了测量固体密度的两种基本方法……
    2. 直接测量法用多次直径测量取平均的做法……
    3. 本次实验也让我对误差来源有了更全面的认识……
