# Fake LLM Plugin

A dummy LLM plugin for Mogan that uses LLM styling but provides simple echo functionality.

## Structure

- `progs/init-llm.scm` — Plugin initialization
- `goldfish/tm-llm.scm` — REPL script (echo mode)
- `packages/session/llm.ts` — Session style package

## Usage

Open **Insert -> Session -> LLM** to start a fake LLM session.
