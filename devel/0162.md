## 2026/05/27 实现假的 llm 插件

### What
实现一个假的 llm 插件，功能上类似 echo 插件（简单响应），但样式使用 stem_202602 中 llm 插件的样式。

### Why
用于在墨干仓库开发 llm 插件相关功能，避免依赖真实的 LLM API。

### How
1. 复制 stem_202602 中 llm 插件的样式文件 `packages/session/llm.ts`
2. `init-llm.scm` 参考 echo 插件的初始化逻辑，但注册为 llm 插件
   - 使用 `goldfish` 作为解释器运行 `tm-llm.scm`
   - 优先加载用户目录下的脚本，回退到系统目录
   - 配置 `llm-serialize` 序列化器和 `llm-launcher` 启动器
3. `tm-llm.scm` 基于 echo 的 read-eval-print 逻辑，主要改动：
   - prompt 改为 `llm>`
   - 增加 `echo-count` 计数器，输出当前次数和输入字符长度
   - 使用 `flush-verbatim` 输出响应
4. 新增 `README.md` 说明插件结构和使用方式
