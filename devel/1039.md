# [1039] 移除 Chat 切换对话模式时的 Spacer 高度动画

## 1 相关文档
- [dddd.md](dddd.md) - 任务文档模板

## 2 任务相关的代码文件

## 3 如何测试

### 3.1 确定性测试（单元测试）
```
xmake b stem
```

### 3.2 非确定性测试（文档验证）
在 macOS 上启动 Mogan STEM，打开 Chat Tab，发送一条消息触发对话模式切换，确认窗口不再被拉大。

## 4 如何提交

提交前执行以下最少步骤：

```bash
xmake b stem
```

## 5 What
移除 ChatConversationPanel::enterConversationMode 中 topSpacer 的高度渐变动画，改为瞬时切换。

## 6 Why
QVariantAnimation 驱动 spacer 高度变化时，每帧反复调用 layout()->invalidate() / layout()->activate()，导致 macOS 上窗口响应内容尺寸变化而被自动拉大。

## 7 How
将原来的 QVariantAnimation 替换为直接调用 topSpacer_->changeSize() 一次设置目标高度。
