2. 确认会话标题背景透明，文字颜色为黑色（浅色主题）或白色（深色主题）

4. 确认标题居中显示，无圆角背景

## 5 What
优化 Chat 侧边栏顶部会话标题（session title）的视觉样式。

1. 字体大小从 13px（kNavButtonFontPx）增大到 18px，新增常量 kSessionTitleFontPx
2. 浅色主题下文字颜色从 #888888 改为 #000000，背景从 #f0f0f0 改为 transparent
3. 深色主题下文字颜色从 #aaaaaa 改为 #ffffff，背景从 #3a3a3a 改为 transparent
4. 移除内联样式中的 padding、border-radius、minHeight 设置，样式完全由 CSS 控制
5. 清理不再使用的常量 kModelLabelMinHeight 和 kModelLabelRadius

## 6 Why
会话标题作为页面主标题，之前使用导航按钮级别的 13px 字体和灰色文字，视觉层级不够突出。标题应使用更大的字号和更鲜明的文字颜色（黑/白），背景透明以融入整体界面。

## 7 How
- 在 qt_chat_tab_widget.cpp 中新增 constexpr int kSessionTitleFontPx = 18，替换原来的 kNavButtonFontPx
- 移除 sessionTitle_ 的 setStyleSheet 和 setMinimumHeight 调用，样式完全交给 liii.css / liii-night.css 中的 QLabel#chat-tab-model-label 选择器控制
- 清理仅在 sessionTitle_ 中使用的 kModelLabelMinHeight 和 kModelLabelRadius 常量
