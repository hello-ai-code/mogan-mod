## 5 What
为暗色主题下的聊天按钮提供专用图标，并将发送按钮从 QPushButton 改为 QToolButton 以支持 CSS `qproperty-icon` 控制图标。

1. 新增 send-dark.svg、cancel-dark.svg、thinking-dark.svg 三个暗色图标资源
2. 将 sendButton 从 QPushButton 改为 QToolButton，图标通过 CSS `qproperty-icon` 控制
3. 使用动态属性 `generating` 配合 CSS 属性选择器实现 Idle/Generating 状态图标切换
4. 优化暗色主题下思考按钮的视觉样式（边框、背景、图标颜色）

## 6 Why
暗色主题下发送按钮、取消按钮、思考按钮仍使用亮色图标（白色/深色），导致图标与深色背景不协调，视觉效果差。需要为暗色主题提供专属图标，并建立主题感知的图标切换机制。

## 7 How
- QPushButton 不支持 CSS `qproperty-icon` 属性，因此将 sendButton 改为 QToolButton
- Idle/Generating 状态切换通过 `setProperty("generating", true/false)` 设置动态属性，配合 `unpolish/polish` 刷新样式，CSS 用 `[generating="true"]` 选择器匹配对应图标
- 暗色图标使用 `#2791ad` 色值，与暗色主题的整体色调保持一致
