## 5 What

将 `qt_chat_tab_widget.cpp` 中所有硬编码的颜色和内联 CSS 提取到 `liii.css`（亮色主题）和 `liii-night.css`（暗色主题），新增 `LLM Chat Tab` 分类，使 Chat Tab 支持主题切换。

1. **移除硬编码颜色**：
   - 移除 `sidebar->setStyleSheet("background-color: #f5f5f5;")` 等所有内联颜色设置
   - 移除 `area->viewport()->setStyleSheet("background-color: #ffffff;")` 等视口颜色
   - 移除所有 `background-color: #xxxxxx`、`color: #xxxxxx` 等硬编码值
   - 保留仅与 DPI 缩放相关的布局属性（`border-radius`、`padding` 等）

2. **新增 CSS 选择器和样式**（亮色主题 `liii.css`）：
   - `QWidget#chat-tab-sidebar`、`QWidget#chat-tab-scroll-content`、`QWidget#chat-tab-conversation-list`、`QWidget#chat-tab-archive-list` → `#f5f5f5`
   - `QWidget#chat-tab-header`、`QWidget#chat-tab-sidebar-normal`、`QWidget#chat-tab-multi-select-bar` → `#f5f5f5`
   - `QWidget#chat-tab-content`、`QWidget#chat-tab-conversation-page` → `#ffffff`
   - `QPushButton#chat-tab-new-btn` → `background: #ffffff, color: #333333`
   - `QPushButton#chat-tab-conversation-btn` → `background: #ffffff, border: #d9d9d9, checked: #e8eefc`
   - `QPushButton#chat-tab-collapse-btn` → `background: #e8e8e8, hover: #d0d0d0`
   - `QPushButton#chat-tab-send-btn` → `hover: #f0f0f0, pressed: #e0e0e0`
   - `QCheckBox#chat-tab-select-checkbox` → `checked: #4a90d9, unchecked: #ffffff`
   - 等等

3. **新增 CSS 选择器和样式**（暗色主题 `liii-night.css`）：
   - 所有选择器加 `QWidget#centralWidget` 前缀以覆盖全局规则 `QWidget#centralWidget QWidget { background: #202020 }`
   - `QWidget#chat-tab-sidebar` 及其子控件 → `#2c2c2c`
   - `QWidget#chat-tab-content`、`QWidget#chat-tab-conversation-page` → `#202020`
   - `QPushButton#chat-tab-new-btn` → `background: #3a3a3a, color: #ffffff`
   - `QPushButton#chat-tab-conversation-btn` → `background: #3a3a3a, border: #5c5c5c, checked: #1a3a5a`
   - `QPushButton#chat-tab-collapse-btn` → `background: #747474, hover: #616161`
   - `QPushButton#chat-tab-send-btn` → `hover: #3a3a3a, pressed: #4c4c4c`
   - `QCheckBox#chat-tab-select-checkbox` → `checked: #4a90d9, unchecked: #2c2c2c`
   - 等等

4. **给控件补充 objectName**：
   - `scrollContent` → `chat-tab-scroll-content`
   - `headerWidget` → `chat-tab-header`

5. **提取通用布局属性**：
   - `text-align: center`（new-btn）
   - `text-align: left`（archive-header、conversation-btn）
   - `font-weight: bold`（archive-header）
   - `font-weight: 600`（conversation-btn:checked）

## 6 Why

1. `qt_chat_tab_widget.cpp` 中存在大量硬编码颜色值，导致暗色主题下 Chat Tab 显示异常（白色背景在暗色主题中刺眼）
2. 项目已有 `liii.css` / `liii-night.css` 主题系统，Chat Tab 应当复用
3. 全局规则 `QWidget#centralWidget QWidget { background: #202020 }` 会强制覆盖子控件背景，暗色主题下必须通过更高优先级选择器覆盖

## 7 How

1. 搜索 `qt_chat_tab_widget.cpp` 中所有 `setStyleSheet` 调用，逐条分析是否包含颜色值
2. 颜色值迁移到 CSS 文件，使用 `#objectName` 精确选择器匹配
3. 暗色主题中所有涉及 `background` 的选择器必须加 `QWidget#centralWidget` 前缀，因为该文件第 20 行有全局覆盖规则
4. 部分容器（如 `scrollContent`、`headerWidget`）在 C++ 中补充 `setObjectName`，以便 CSS 精确选择
5. `border: 1px solid #cccccc` 等含颜色的简写 border 被拆分为 `border-width: 1px; border-style: solid`，颜色部分移到 CSS
