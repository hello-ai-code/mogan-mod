# [1069] 修复 Windows send 按钮样式出错的问题

## 1 相关文档
- [dddd.md](dddd.md) - 任务文档模板

## 2 任务相关的代码文件
- `src/Plugins/Qt/qt_chat_tab_widget.cpp` - 聊天面板 send 按钮样式

## 3 如何测试

### 3.2 非确定性测试（UI 验证）
```
xmake b stem
xmake r stem
```

1. 打开 AI Chat 面板，确认 send 按钮大小正常（不被重复 DPI 缩放）
2. 确认 send 按钮圆角样式只作用于发送按钮，不影响其他 QToolButton

## 4 如何提交

提交前编译项目：

```bash
xmake b stem
```

## 5 What

1. 修复 `setFixedSize` 参数对 DPI 缩放值再次调用 `DpiUtils::scaled` 导致双重缩放的问题
2. 将 stylesheet 选择器从 `QToolButton` 改为 `QToolButton#chat-tab-send-btn`，避免样式泄漏到其他 QToolButton

## 6 Why

Windows 下 send 按钮图标过大。`sendIconSize` 已经是 `DpiUtils::scaled` 后的值，`setFixedSize` 又对该值再调一次 `DpiUtils::scaled`，导致高 DPI 屏幕上按钮被放大两倍。同时未限定的 `QToolButton` 选择器会影响面板中其他工具按钮的样式。

## 7 How

`setFixedSize` 直接传入 `sendIconSize`（已是缩放后的值），stylesheet 选择器加上 `#chat-tab-send-btn` 限定作用域。
