# [216_31] 启动页代码优化与 Scheme 字符串转义统一

## 如何测试

### 前置步骤
1. 编译：`xmake b stem`
2. 启动 Mogan，进入启动页。

#### 1. File 页面卡片交互
- 打开启动页 **File** 页面，Hover "New Blank Document" 卡片。
- **预期**：仅 CSS `:hover` 边框变色，无选中边框、无异常输出。
- 点击卡片，确认可正常新建空白文档。

#### 2. Recent 文档增删与 Scheme 转义（重点）
- **macOS / Linux**：创建文件名含双引号的文档（如 `"test".tmu`），打开后返回启动页，确认：
  - 该文档出现在 **Recent** 列表。
  - 点击该文档可正常打开。
  - 右键删除该项，确认从列表消失。
- **Windows**：创建路径含反斜杠的文档（如 `C:\Users\test\doc.tmu`），重复上述增删查操作。
- **中文路径**：创建文件名含中文的文档（如 `毕业论文大删减版.tmu`），打开后返回启动页，确认：
  - 该文档出现在 **Recent** 列表。
  - 点击该文档可正常打开，**不应**出现 `The file or buffer does not exist` 报错。
  - 右键删除该项，确认从列表消失。
- **回归验证**：打开任意已有文档，返回启动页，确认该文档被置顶到 Recent 列表首位。

#### 3. Template 页面打开文档
- 进入 **Template** 页面，下载并打开一个模板。
- **预期**：模板正常加载，无 Scheme 解析错误。

#### 4. Recent 列表内存泄漏回归
- 启动程序，确保 Recent 列表有若干项。
- 在 File 页面和 Template 页面之间反复切换 20 次以上（每次切换回 File 页会触发 `refreshRecentDocs`）。
- **预期**：程序内存占用稳定，无持续增长。

## 2026/04/23 实现说明

### What

**src/Plugins/Qt/qt_file_page.cpp**
- 修复 `renderRecentDocs()` 内存泄漏：`QListWidget::clear()` 不释放 `setItemWidget` 关联的 widget，改为手动 `takeItem` + `delete`。
- 删除 `StyleCard` 冗余状态：`setSelected()`、`isSelected_`、`selectedCard_`、`hovered` 信号、`enterEvent`。
- 简化 `paintEvent`：移除 `QPainter::Antialiasing` 和手动选中边框绘制。
- 提取 `populateRecentDocsFromPaths()` 消除重复代码。
- `createDocumentWithStyle()` 改用 `qt_scheme_quote()`，统一 Scheme 字符串转义。
- 文件路径相关的 Scheme 调用（`addRecentDoc`、`removeRecentDoc`、`onRecentDocClicked`）改用 `qt_scheme_quote_utf8()`，**避免中文路径被 Cork 编码转义**（如 `<#6BD5><#4E1A>...`），否则操作系统文件系统无法识别，导致 `The file or buffer does not exist` 报错。

**src/Plugins/Qt/qt_file_page.hpp**
- 移除 `hovered` 信号、`enterEvent` 声明、`isSelected_` / `selectedCard_` / `badgeLabel_` 成员。

**src/Plugins/Qt/qt_startup_tab_widget.cpp / .hpp**
- 删除内联 `escape_for_scheme_string()`，键盘事件改用 `qt_scheme_quote()`。
- Template 页面打开文档改用 `qt_scheme_quote_utf8()`，确保中文模板路径正确传递。
- `Entry::OpenFolder` 重命名为 `Entry::OpenDocument`，与按钮文案一致。

**src/Plugins/Qt/qt_utilities.cpp / .hpp**
- 新增 `qt_scheme_quote()`，底层调用 `moebius::data::scm_quote()`。
- 新增 `qt_scheme_quote_utf8()`，底层调用 `moebius::data::scm_quote(from_qstring_utf8(text))`。
- 将实现从文件末尾移至 `qt_translate()` 附近。

**src/Plugins/Qt/qt_template_page.cpp**
- `onTemplatesLoaded()` 中补充 `categoryBar_->layout()` 空检查。
