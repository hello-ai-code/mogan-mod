## What

将 `margins` 相关术语的翻译从 "边白" 统一优化为 "边距"，提升表达自然度和术语准确性。

具体改动：

**修正现有翻译：**
1. `("margins" "边白")` → `("margins" "页边距")`
2. `("margins on paper" "纸上边白")` → `("margins on paper" "页边距")`
3. `("margins on screen" "屏幕边白")` → `("margins on screen" "屏幕边距")`
4. `("margins as on paper" "显示纸面边白")` → `("margins as on paper" "显示页边空白")`
5. `("reduced margins" "占用边白")` → `("reduced margins" "占用页边空白")`
6. `("same screen margins as on paper" "保持屏幕边白和纸上边白一致")` → `("same screen margins as on paper" "保持屏幕边距和页边距一致")`

**新增翻译：**
7. `("Determine margins from text width" "按文本宽度自动确定边距")`
8. `("Toggle mode for using reduced margins to save paper" "切换窄边距模式以节省纸张")`
9. `("Toggle mode for using special margins for screen editing" "切换屏幕编辑专用边距模式")`
10. `("Toggle mode for using standard page numbering" "切换是否显示页码")`

## Why

"边白" 是源自日文排版术语的译法，在现代中文软件界面中并不常见。中文用户更熟悉的标准术语是 **"页边距"**（对应 CSS / Word 等软件的 margin 翻译）。统一术语可以降低用户理解成本，使界面更符合中文软件惯例。

## How

在 `TeXmacs/plugins/lang/dic/en_US/zh_CN.scm` 中搜索所有包含 "边白" 的条目，根据上下文替换为 "边距" 或 "页边距"。注意保持前后一致，避免同一概念出现两种译法。
