# 216_17 移除启动页设置中未生效的占位功能

## 如何测试
1. 编译：`xmake b stem`
2. 打开启动页 `Settings` 页面，确认不再显示 "Startup" 标签页。
3. 确认设置页面直接从 "General" 标签页开始，其他标签页（Keyboard, Mathematics, Convert, Other）正常显示。
4. 确认原有设置功能不受影响。

## 2026/04/17 实现说明

### What
本次移除启动页设置中未实际生效的占位功能：
1. 删除 "Startup" 专属设置标签页。
2. 移除以下未生效的设置项：
   - Default view（默认视图选择：Recent files / Home / Help）
   - Show file thumbnails（显示文件缩略图）
   - Group by date（按日期分组）

#### 修改文件

**TeXmacs/progs/startup-tab/startup-tab-settings.scm** (修改)
- 删除 `;; Startup tab specific settings` 整个区块。
- 删除 `define-preference-names` 定义的 "startup-tab:default-view" 枚举值。
- 删除 `startup-tab-preferences-widget` 定义。
- 在 `startup-settings-widget` 中删除 "Startup" 标签页引用。
- 设置页面现在直接从 "General" 标签页开始。

### Why
1. 这些设置项在代码中只是占位符，实际功能并未实现。
2. "Default view" 切换视图的功能不存在。
3. "Show file thumbnails" 和 "Group by date" 的 UI 开关未与任何实际逻辑绑定。
4. 保留未生效的设置会误导用户，降低体验。

### How
1. 直接删除整个 `startup-tab-preferences-widget` 及其相关定义。
2. 从 `startup-settings-widget` 的 `icon-tabs` 中移除对应的标签页入口。
3. 保留其他设置标签页（General, Keyboard, Mathematics, Convert, Other）不受影响。
