# 216_37 启动页 liii-night 主题背景色修复及白线消除

## 如何测试

1. 编译：`xmake b stem`
2. 启动 Mogan，切换到 **liii-night** 暗色主题。
3. 打开启动页，进入 **File** 页面：
   - 确认页面整体背景为 **#f3f3f3**（浅灰，区别于内容编辑区的深色）。
   - 确认文档样式卡片容器背景为 **#2c2c2c**，与暗色主题协调。
4. 切换至 **Template** 页面：
   - 确认页面整体背景为 **#f3f3f3**。
   - 确认分类按钮栏（Category Bar）背景为 **#2c2c2c**。
   - 确认模板卡片（含缩略图+标题）整体容器背景为 **#2c2c2c**。
5. 切换回 **liii** 亮色主题，重复上述步骤，确认无样式异常。
6. 启动页顶部（标签页栏与内容区之间）确认 **无白线**。

## 2026/04/30 实现说明

### What

修复 `liii-night` 暗色主题下启动页 File / Template 页面背景色不统一、部分容器过亮，以及启动页顶部 1px 白线问题。

#### 修改文件

**src/Plugins/Qt/qt_startup_tab_widget.cpp**
- `create_file_page()`：为 `filePage_` 设置 `objectName("startup-tab-file")`。
- `create_template_page()`：为 `templatePage_` 设置 `objectName("startup-tab-template")`。

**src/Plugins/Qt/qt_file_page.cpp**
- `setupStyleCards()`：为 `cardsContainer_` 设置 `objectName("startup-tab-file-cards")`。

**src/Plugins/Qt/qt_template_page.cpp**
- `setupUI()`：为 `categoryBar_` 设置 `objectName("startup-tab-category-bar")`。

**src/Plugins/Qt/qt_tm_widget.cpp**
- `qt_tm_widget_rep` 构造函数：将 `centralWidget` 主布局的 `setContentsMargins` 从 `(0, 1, 0, 0)` 改为 `(0, 0, 0, 0)`，消除顶部 1px 白线。

**TeXmacs/misc/themes/liii-night.css**
- 新增 `QWidget#centralWidget QWidget#startup-tab-file`、`QWidget#centralWidget QWidget#startup-tab-template`：背景色 `#f3f3f3`。
- 新增 `QWidget#centralWidget QWidget#startup-tab-file-cards`、`QWidget#centralWidget QWidget#startup-tab-category-bar`、`QWidget#centralWidget QWidget#startup-tab-template-item`：背景色 `#2c2c2c`。

### Why

1. **暗色主题一致性**：`liii-night` 下启动页右侧内容区默认继承 `#202020`，但 File / Template 页面作为独立功能页，需要在暗色主题下保持统一的浅灰底（`#f3f3f3`），与左侧导航栏深色形成对比，同时避免过暗导致文字和卡片辨识度下降。
2. **容器背景修复**：样式卡片容器、分类栏、模板卡片项在暗色主题下之前透出 `centralWidget` 的 `#ffffff` 或 `#202020`，导致色块断层；显式设置为 `#2c2c2c` 后与暗色主题其他面板保持一致。
3. **白线根因**：`qt_tm_widget.cpp` 中 `centralWidget` 布局设置了 1px 顶部 margin，在启动页模式下会透出 `centralWidget` 的白色背景，形成一条肉眼可见的白线。去掉该 margin 后，白线消失，且不影响编辑器模式（编辑器画布由 `tm_background` 独立控制）。

### How

- 为 File / Template 页面及其子容器添加唯一 `objectName`，使 CSS 可以通过高优先级选择器精确控制背景色，而不影响其他 `QWidget`。
- `liii-night.css` 中使用 `QWidget#centralWidget QWidget#xxx` 两级选择器提升优先级，覆盖通用的 `QWidget#centralWidget QWidget` 规则。
- 直接移除 `centralWidget` 布局的顶部 1px margin，不通过修改 CSS 背景色来"掩盖"白线，根因修复更干净。
