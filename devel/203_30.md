# [203_30] 新增复制到 LaTeX 快捷键


## 2026/04/28 如何测试

1. 在文档中选中一段普通文本或数学公式。
2. 在 Windows/Linux 上按 `Ctrl+Shift+L`；在 macOS 上按 `Command+Shift+L`。
3. 粘贴到纯文本编辑器，确认剪贴板内容为 LaTeX。
4. 在 Scheme/headless 环境中确认 `std L` 绑定到：

```scheme
(clipboard-copy-export "latex" "primary")
```
## 新增复制到 LaTeX 快捷键

### What

新增平台标准全局快捷键，用于将当前选中的内容复制为 LaTeX：

- Windows/Linux: `Ctrl+Shift+L`
- macOS: `Command+Shift+L`

本次涉及：

- `TeXmacs/progs/generic/generic-kbd.scm`

### Why

此前用户需要通过菜单 `Edit -> Copy to -> LaTeX` 才能把选区复制为 LaTeX，键盘操作不够直接。

### How

在通用键盘映射中加入：

```scheme
("std L" (clipboard-copy-export "latex" "primary"))
```

其中 `std L` 在 Windows/Linux 上表示 `Ctrl+Shift+L`，在 macOS 上表示 `Command+Shift+L`。该命令复用已有的 `clipboard-copy-export` 逻辑，与菜单中的 `Copy to -> LaTeX` 走同一条导出路径。

同时引入 `(utils edit selections)`，保证键盘模块可以直接使用 selection 导出工具。

原先 `std L` 绑定到 `(open-document*)`，会与该快捷键冲突；本次将 `std L` 改为复制到 LaTeX，保留 `std o` 作为打开文档入口。
