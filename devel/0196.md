# [0196] 实现 gf format 工具替代 bin/format

## 1 相关文档
- [goldfish gfproject.json](https://github.com/LiiiLabs/goldfish/blob/main/gfproject.json) - 工具注册配置

## 2 任务相关的代码文件
- `gfproject.json` - 注册 format/fmt 工具
- `tools/format/liii/goldformat.scm` - gf format 命令实现

## 3 如何测试
```bash
gf format
```

## 4 如何提交
提交前执行以下最少步骤：

```bash
gf fmt --changed-since=main
```

## 5 What
在 gfproject.json 中注册 `format` 工具，实现 `gf format` 命令替代 `bin/format` 脚本。

## 6 Why
`bin/format` 是 elvish 脚本，依赖特定 shell。`gf format` 基于 Goldfish Scheme 实现，跨平台且与 `gf` 工具链统一。

## 7 How
1. 创建 `gfproject.json`，注册 `format`（goldformat 模块）和 `fmt`（goldfmt 模块）
2. 实现 `tools/format/liii/goldformat.scm`：
   - 递归收集 `tests/`、`src/`、`moebius/`、`3rdparty/lolly/` 下的 C++ 文件
   - 将文件列表写入临时文件，通过 `clang-format --files=` 一次性调用格式化
   - 逐目录调用 `gf fmt` 格式化 Scheme 文件
   - 自动检测 `clang-format` 路径（Linux/macOS/Windows）
