# 216_24 暗色模式启动页勾选框样式修复

## 如何测试
1. 编译：`xmake b stem`
2. 启动 Mogan，确保主题为 "Liii Dark"（暗色模式）。
3. 打开启动页，切换到 "Settings" 页面。
4. 确认所有勾选框（如 "Use large brackets", "Show full context" 等）背景色可见，不是透明的。
5. 勾选和取消勾选，确认勾选框的勾选标记清晰可见。
6. 测试其他页面的勾选框（如 Convert -> Html 中的选项）。
7. 切换主题到 "Liii"（亮色模式），确认勾选框显示正常（无异常变色）。
8. 切换主题到 "Default" 或 "Native"，确认勾选框使用系统默认样式。

## 2026/04/21 实现说明

### What
修复暗色模式下启动页勾选框（QCheckBox）背景不可见的问题。

#### 修改文件

**TeXmacs/misc/themes/liii-night.css** (修改)
- **调整选择器顺序**：
  - 将 `QWidget#centralWidget QWidget` 从文件末尾（原第967-970行）移动到文件开头（第17-20行）。
  - 这是为了给后续更具体的控件样式提供"基线"样式。
- **添加勾选框样式**：
  - 新增 `QWidget#centralWidget QCheckBox` 选择器，设置 `background-color: #6a6a6a`。
  - 由于该选择器定义在 `QWidget#centralWidget QWidget` 之后，且两者优先级相同（101），后定义的规则生效。
- **简化样式规则**：
  - 删除原来复杂的 `QWidget#centralWidget QWidget#startup-tab-content QCheckBox` 选择器。
  - 删除配套的 `QRadioButton` 样式（如有需要可后续单独添加）。

**修改前的样式（已删除）：**
```css
QWidget#centralWidget QWidget#startup-tab-content QCheckBox,
QWidget#centralWidget QWidget#startup-tab-content QRadioButton {
  background-color: #6a6a6a;
  border: 1px solid #8a8a8a;
  border-radius: 2px;
}
```

**修改后的样式：**
```css
/* 文件开头 - 基线样式 */
QWidget#centralWidget QWidget {
  background: #202020;
  color: #ffffff;
}

/* 后面覆盖勾选框样式 */
QWidget#centralWidget QCheckBox {
  background-color: #6a6a6a;
}
```

### Why
1. **问题原因**：
   - `QWidget#centralWidget QWidget` 为编辑区域设置了深色背景（#202020）。
   - 启动页设置页面的勾选框是通过 Scheme 动态创建的，它们的父控件链包含 `QWidget#centralWidget`。
   - 该选择器优先级为 101（100 for ID + 1 for 类型），会覆盖优先级较低的 `QCheckBox` 样式。

2. **CSS 优先级冲突**：
   - `QWidget#centralWidget QWidget` = 101
   - `QWidget#centralWidget QCheckBox` = 101
   - 当优先级相同时，**后定义的样式获胜**。
   - 原来 `QWidget` 样式在文件末尾，导致任何前面的 `QCheckBox` 样式都被覆盖。

3. **原解决方案的问题**：
   - 使用 `QWidget#centralWidget QWidget#startup-tab-content QCheckBox` 虽然优先级更高（201），但过于具体，难以维护。

### How
1. **调整文件结构**：
   - 将通用的 `QWidget#centralWidget QWidget` 样式移到文件开头，作为基线。
   - 在后面定义具体的控件覆盖样式（如 `QCheckBox`、`QPushButton` 等）。

2. **利用 CSS 层叠规则**：
   - 相同优先级的选择器，后定义的覆盖先定义的。
   - `QCheckBox` 定义在 `QWidget` 之后，因此其 `background-color` 生效。

3. **影响范围**：
   - 此修改会影响所有在 `QWidget#centralWidget` 内的 `QCheckBox`。
   - 包括启动页设置页面、以及其他动态创建的勾选框。
