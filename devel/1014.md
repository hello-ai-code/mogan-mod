## 5 What

1. **TemplateAPI 新增推荐模板接口**：新增 `fetchRecommendTemplates()` 调用 `POST /api/v1/doc/template/recommend`，新增对应信号 `recommendTemplatesLoaded` / `recommendTemplatesLoadFailed`
2. **TemplateManager 管理推荐模板**：新增 `refreshRecommendTemplates()` 在 `initialize()` 中自动触发，新增 `recommendTemplates()` 按 API 返回顺序提供列表，维护 `recommendTemplateIds_` 记录推荐顺序
3. **QTMHomePage 移除硬编码模板**：移除 `elegantbook` 和 `nsfc-ysf-c` 两个硬编码模板，改为连接 `recommendTemplatesLoaded` 信号动态创建卡片
4. **parseTemplatesResponse 兼容两种格式**：推荐接口返回 `data` 为数组，列表接口返回 `data.items` 为数组，解析逻辑同时兼容
5. **防重复请求**：通过 `recommendTemplatesFetched_` 标记确保每个会话只请求一次推荐模板

## 6 Why

1. **硬编码模板不可扩展**：此前 homepage 写死了两个模板，新增或替换模板需要改代码发版
2. **服务端可控推荐内容**：运营团队可以通过 API 随时调整首页展示的推荐模板，无需客户端更新
3. **与 TemplateCenter 体系统一**：推荐模板和分类模板共用同一套 metadata 结构和缓存机制，减少重复代码

## 7 How

### 7.1 请求流程

```
[启动]
  |
  v
[TemplateManager::initialize]
  |
  +-- 初始化缓存
  +-- 加载本地分类
  +-- 加载缓存模板
  +-- 后台请求远程分类
  +-- 后台请求推荐模板 (新增)
        |
        v
  [TemplateAPI::fetchRecommendTemplates]
        POST /api/v1/doc/template/recommend
        |
        v
  [onRecommendTemplatesReplyFinished]
        解析响应，更新 recommendTemplateIds_
        mergeMetadata() 合并到 templates_
        保存缓存
        emit recommendTemplatesLoaded()
        |
        v
  [QTMHomePage::onRecommendTemplatesLoaded]
        refreshTemplateCards()
        动态创建 StyleCard
        加载缩略图
```

### 7.2 防重复请求机制

```cpp
void TemplateManager::refreshRecommendTemplates () {
  if (isRefreshingRecommendTemplates_ || recommendTemplatesFetched_) {
    return;  // 正在请求中，或已成功获取过
  }
  isRefreshingRecommendTemplates_= true;
  api_->fetchRecommendTemplates ();
}

void TemplateManager::onRemoteRecommendTemplatesLoaded (...) {
  isRefreshingRecommendTemplates_= false;
  recommendTemplatesFetched_     = true;  // 标记已完成，后续不再请求
  ...
}
```

### 7.3 响应格式兼容

推荐接口和列表接口的数据结构不同：

| 接口 | 响应格式 | 解析方式 |
|---|---|---|
| `/api/v1/doc/template/recommend` | `{ "code": 0, "data": [...] }` | `data` 直接为数组 |
| `/api/v1/doc/template/list` | `{ "code": 0, "data": { "items": [...] } }` | `data.items` 为数组 |

```cpp
QJsonArray array;
if (data.isArray ()) {
  array= data.toArray ();           // 推荐接口
}
else if (data.isObject ()) {
  array= data.toObject ().value ("items").toArray ();  // 列表接口
}
```

### 7.5 关键状态变量

| 变量 | 含义 |
|---|---|
| `isRefreshingRecommendTemplates_` | 是否正在请求推荐模板（防重复） |
| `recommendTemplatesFetched_` | 是否已成功获取过推荐模板（本会话只请求一次） |
| `recommendTemplateIds_` | 推荐模板的 ID 列表，保持 API 返回顺序 |
