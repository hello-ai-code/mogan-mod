# [0643] 添加 Border color 的中文翻译

## 1 相关文档
- [dddd.md](dddd.md) - 任务文档模板
- [0643.md](0643.md) - 本任务文档

## 2 任务相关的代码文件
- `TeXmacs/plugins/lang/dic/en_US/zh_CN.scm`

## 3 如何测试

### 3.1 确定性测试（单元测试）
无新增单元测试。本次改动为翻译字典条目补充。

### 3.2 非确定性测试（文档验证）
检查 `TeXmacs/plugins/lang/dic/en_US/zh_CN.scm` 文件中已包含以下条目：
```scheme
("Border color" "边框颜色")
```

启动 Mogan STEM 后，按 `ALT` + `t`插入一个表格，在界面中出现 "Border color" 文本的位置（如表格或单元格边框相关设置），应显示为中文 "边框颜色"。

## 4 如何提交

提交前执行以下最少步骤：

```bash
gf fmt --changed-since=main
```

## 5 What
在简体中文翻译字典 `zh_CN.scm` 中新增 `"Border color"` 到 `"边框颜色"` 的映射。

## 6 Why
Mogan STEM 界面中的 `"Border color"` 在简体中文环境下尚未被翻译，会显示为英文。补充该翻译项可提升中文用户的使用体验。

## 7 How
在 `TeXmacs/plugins/lang/dic/en_US/zh_CN.scm` 中按字典字母顺序，在 `"Bitmap export resolution (dpi):"` 与 `"Braces { }"` 之间插入：

```scheme
("Border color" "边框颜色")
```
