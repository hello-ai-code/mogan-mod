# [321] fix: Alt+W (M-w) should copy like Emacs

## What
Alt+W (M-w) was not working as copy in Emacs mode.

## Why
The keybinding was mapped to "M-W" (capital W) instead of 
"M-w" (lowercase w), causing Alt+W to not trigger the copy action.

## How
Changed "M-W" to "M-w" in prefix-kbd.scm to correctly map 
Alt+W to the copy action in Emacs mode.

## How to test
1. Open Mogan
2. Enable Emacs mode
3. Select some text
4. Press Alt+W
5. The text should now be copied (not cut)
