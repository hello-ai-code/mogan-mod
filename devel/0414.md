## 3 如何测试

### 3.1 手动更改版本（模拟测试环境）
在 `TeXmacs/progs/doc/help-funcs.scm` 的 `(mogan-version)` 中，修改 `cur-ver` 的绑定即可模拟不同版本状态：
- **版本一致测试**：保持 `cur-ver` 为 `(xmacs-version)` 或手动写死为 `"2026.2.5"`（与 `latest-ver` 相同）。
- **版本不一致测试**：将 `cur-ver` 临时改写为较低版本，如 `"2026.2.4"`。

### 3.2 验证步骤
1. **测试默认语言（英文）**：
   - 将系统或界面语言设置为 “English”（或任何未翻译的语言，如德语等）。
   - 验证版本一致和不一致时的提示信息，确认显示为默认的英文文本：
     - 一致：`You are using vX.Y.Z, and the latest stable version is vA.B.C.`
     - 不一致：`You are using vX.Y.Z, and the latest stable version is vA.B.C. Please click OK to visit the official website to download the latest stable version.`
2. **测试简体中文**：
   - 切换界面语言为 “简体中文”。
   - 验证版本一致和不一致时的提示信息是否正确翻译为中文。
3. **测试繁体中文/台湾**：
   - 切换界面语言为 “繁體中文（台灣）”。
   - 验证版本一致和不一致时的提示信息是否正确翻译为繁体中文。

## 5 What
把原先硬编码在 `(mogan-version)` 代码里的简体中文和繁体中文提示词重构为统一的英文词条，并通过 `translate` 和 `replace` 内置翻译函数动态提取翻译，在未找到翻译字典时自动 Fallback 到默认的英文提示。

## 6 Why
1. **解耦业务逻辑与文本展现**：通过国际化系统（i18n）管理多语言文案，避免了在 Scheme 代码中直接写死多语言分支（如 `cond ((== lan "chinese") ...)`），让核心业务逻辑更加纯粹、易维护。
2. **便于扩展多语言**：将词条移入国际化运行词典后，其他语言（如法语、日语等）翻译人员只需要在对应的字典文件中追加英文对应的翻译即可，不需要修改 `.scm` 逻辑源码。
3. **保证默认 fallback 的健壮性**：即使没有针对某些非中文语种的词条字典，版本提示信息也会默认显示规范的英文文案，不会由于词条未匹配而出现空白或错误。

## 7 How
1. **重构版本函数**：
   在 `TeXmacs/progs/doc/help-funcs.scm` 中，简化 `(mogan-version)` 函数：
   - 删除了原本读取输出语言并对 `chinese`/`taiwanese` 进行专门 `cond` 分流的分支代码。
   - 使用 `(translate "Version")` 获取本地化标题。
   - 使用 `(replace "You are using v%1..." cur-ver latest-ver)` 传入带参数的英文源文本词条，使其动态根据当前语言环境翻译并填充版本变量。
2. **注册及更新简体中文运行翻译**：
   为了确保运行时的 C++ 翻译模块能够正确加载该词条，我们在主运行词典 `TeXmacs/plugins/lang/dic/en_US/zh_CN.scm` 中分别注册了：
   ```scheme
   ("Version" "版本")
   ("You are using v%1, and the latest stable version is v%2." "您正在使用 v%1，当前最新稳定版是 v%2。")
   ("You are using v%1, and the latest stable version is v%2. Please click OK to visit the official website to download the latest stable version." "您正在使用 v%1，当前最新稳定版是 v%2。请点击确认前往官网下载最新稳定版。")
   ```
3. **注册及更新繁体中文运行翻译**：
   我们在繁体中文运行词典 `TeXmacs/plugins/lang/dic/en_US/zh_TW.scm` 中进行了对应的词条补全和更新：
   ```scheme
   ("Version" "版本")
   ("You are using v%1, and the latest stable version is v%2." "您正在使用 v%1，目前最新穩定版是 v%2。")
   ("You are using v%1, and the latest stable version is v%2. Please click OK to visit the official website to download the latest stable version." "您正在使用 v%1，目前最新穩定版是 v%2。請點擊確認前往官網下載最新穩定版。")
   ```
