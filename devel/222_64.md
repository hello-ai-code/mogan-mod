# 222_64 章节编号自定义

## 如何测试

### 功能概述
本功能为章节环境（chapter/section/subsection/subsubsubsection）提供编号样式、分隔符等自定义选项。

### 测试前准备
1. 清理缓存
2. 重新编译
3. 打开测试文档：`TeXmacs/tests/tmu/222_64.tmu`

---

### 一、编号样式设置测试

#### 1.1 支持编号样式的环境
- `chapter`
- `section`
- `subsection`
- `subsubsection`

#### 1.2 不支持编号样式的环境（无编号版本）
- `chapter*`
- `section*`
- `subsection*`
- `subsubsection*`

#### 1.3 编号样式选项验证
对每个支持的环境，测试以下编号样式：
- 阿拉伯数字 (Arabic) - 预期显示：1, 2, 3
- 汉字数字 (Hanzi) - 预期显示：一, 二, 三
- 大写罗马数字 (Roman) - 预期显示：I, II, III
- 小写罗马数字 (roman) - 预期显示：i, ii, iii
- 大写字母 (Alpha) - 预期显示：A, B, C
- 小写字母 (alpha) - 预期显示：a, b, c
- 圆圈数字 (circle) - 预期显示：①, ②, ③

**测试步骤：**
1. 插入对应章节环境
2. 点击小扳手打开首选项菜单
3. 选择 "Number style" → 选择不同样式
4. 观察章节标题编号变化
5. 观察目录中对应章节编号同步变化

---

### 二、编号分隔符设置测试

#### 2.1 编号与标题间隔符号 (section-sep)
- `.` - 预期显示：1. 标题
- `、` - 预期显示：一、标题
- `-` - 预期显示：1- 标题
- `space` - 预期显示：1 标题
- `none` - 预期显示：1标题

#### 2.2 多级编号前缀间隔符号 (prefix-sep)
当 `sectional-short-style` 为 `false` 时测试：
- `.` - 预期显示：1.1. 标题
- `、` - 预期显示：一、一、标题
- `-` - 预期显示：1-1- 标题
- `space` - 预期显示：1 1 标题
- `none` - 预期显示：11标题

**测试步骤：**
1. 插入 section/subsection
2. 点击小扳手 → "Number-title separator" / "Sub-level separator"
3. 选择不同分隔符，观察变化

---

### 三、章节前缀编号测试 (sectional-short-style)

#### 3.1 功能验证
- 勾选 (true) - section 显示独立编号：1, 2, 3
- 不勾选 (false) - section 显示含 chapter 前缀：1.1, 1.2

**测试步骤：**
1. 确保文档有 chapter 环境
2. 插入 section 环境
3. 点击小扳手查看 "Section prefix" 分组
4. 勾选/取消 "Prepend chapter number prefix for section numbers"
5. 观察 section 编号变化：
   - 勾选：显示 "1"（短编号）
   - 取消：显示 "1.1"（长编号，含 chapter 编号前缀）

#### 3.2 与编号样式组合测试
- Hanzi chapter + short-style true - section 也是汉字：一、一、
- Hanzi chapter + short-style false - section 也是汉字：一、一、

---

### 四、显示/隐藏章节编号测试 (section-display-numbers)

#### 4.1 功能概述
允许全局设置是否显示章节编号，支持的环境：
- `chapter`
- `section`
- `subsection`
- `subsubsection`
- `paragraph`
- `subparagraph`

#### 4.2 测试步骤
1. 插入任意章节环境（如 section）
2. 点击小扳手打开首选项菜单
3. 查看对应的分组（根据章节类型显示不同标题）：
   - chapter: "chapter numbering"（全局章编号）
   - section: "section numbering"（全局节编号）
   - subsection: "subsection numbering"（全局小节编号）
   - subsubsection: "subsubsection numbering"（全局子节编号）
4. 勾选/取消 "Global hide * numbers"（全局隐藏*编号）
5. 观察变化：
   - 勾选（true）：章节标题显示编号，如 "1 标题"
   - 取消（false）：章节标题不显示编号，如 "标题"

#### 4.3 各环境默认行为
- 默认显示编号：`chapter`, `section`, `subsection`, `subsubsection`
- 默认不显示编号：`paragraph`, `subparagraph`

### 五、菜单 UI 测试

#### 5.1 菜单结构
- "Number style" 显示为子菜单（→）
- "Number-title separator" 显示为子菜单（→）
- "Sub-level separator" 显示为子菜单（→）

#### 5.2 不可用选项隐藏
- 无编号章节（chapter*/section*等）不显示编号相关选项
- paragraph/subparagraph 不显示编号样式选项
- 不可用选项不应显示为灰色，应完全隐藏

#### 5.3 翻译验证
- preferences for tag -> 全局设置
- rendering options for tag -> 局部设置
- Number-title separator -> 编号与标题间隔符号
- Sub-level separator -> 子层级分隔符
- Section prefix -> 节前缀
- Title style -> 标题样式
- Prepend chapter number prefix for section numbers -> 节编号显示章编号的前缀
- chapter numbering -> 全局章编号
- section numbering -> 全局节编号
- subsection numbering -> 全局小节编号
- subsubsection numbering -> 全局子节编号
- Global hide chapter numbers -> 全局隐藏章编号
- Global hide section numbers -> 全局隐藏节编号
- Global hide subsection numbers -> 全局隐藏小节编号
- Global hide subsubsection numbers -> 全局隐藏子节编号

### 八、Toggle Numbering 交互测试

#### 8.1 功能验证
当全局隐藏编号启用时，点击章节标题旁的 toggle numbering 按钮应弹出提示。

**测试步骤：**
1. 插入 section 环境
2. 点击小扳手，取消勾选 "Global hide section numbers"（隐藏编号）
3. 点击章节标题旁的切换编号按钮（或快捷键 Cmd+#）
4. 观察：编号正常切换（显示/隐藏）
5. 点击小扳手，勾选 "Global hide section numbers"（显示编号）
6. 再次点击切换编号按钮
7. 观察：弹出提示窗口 "Global numbering is hidden, toggle has no effect" / "已设置全局隐藏编号，此处切换不生效"
8. 确认弹窗标题为 "Information" / "提示信息"

**预期行为：**
- 全局隐藏启用时，toggle 不生效并弹出提示
- 全局隐藏未启用时，toggle 正常工作
