# [209_20] 添加 FISH 语言语法高亮

## 如何测试

1. 点击`文档->增加宏包->代码->fish`
2. 选择菜单插入-程序-代码块/行内代码或使用 `\fish` 或者 `\fish-code`插入代码块环境
3. 输入 FISH 代码，检查关键字、字符串、注释、运算符和常见内建函数高亮
``` FISH
; 定义一个函数：计算数组的平均值
define compute_average(arr)
    local sum = 0.0
    local i = 1
    loop while i <= list.size(arr)
        sum = sum + arr(i)
        i = i + 1
    endloop
    return sum / list.size(arr)
end

; 主程序
define main
    local data = list.create
    list.append(data, 10)
    list.append(data, 20)
    list.append(data, 30)

    local avg = compute_average(data)
    io.out("Average value = " + string.build(avg))

    ; 条件判断
    if avg > 15 then
        io.out("Average is greater than 15")
    else
        io.out("Average is 15 or less")
    endif

    ; 使用 math 内建函数
    local x = -3.5
    local y = math.abs(x)
    io.out("Absolute value = " + string.build(y))

    ; command 块（轻量示例）
    command
        model new
        history list
    endcommand
end
```
4. 打开 `TeXmacs/tests/tmu/209_20.tmu` 查看回归样例

## 2026/04/15

### What

为 Mogan 添加 Itasca FISH 语言代码高亮，覆盖核心语句、通用内建命名空间和常见运算符。


- 修正了 FISH 文件格式定义，使用更符合 Itasca 实际使用习惯的 `.fis` / `.FIS` 后缀，而不是 `.fish`。
- 扩充了高亮词表，补充了更多官方通用内建命名空间和函数，并允许带 `_` 和 `.` 的关键字整体参与匹配，因此像 `math.abs`、`io.out`、`string.build`、`version.fish.major` 这类官方 dotted built-ins 可以按整体高亮。
- 细化了语法类别划分，把条件语句、控制语句、类型、模块、成员访问和 splitting 运算符分别映射到更合适的高亮类别，像 `->`、`::`、`=::`、`+=::` 这类 FISH 特有写法的显示效果更准确。
- 优化了 `command ... endcommand` 的处理方式，虽然仍然是词法级近似，但补上了常见命令词的高亮，避免该代码块内部几乎全部退回默认颜色。
- 改进了编辑体验，缩进逻辑不再只看单行前缀，而是同时识别开块、闭块、中间分支和续行场景，对 `if/else/endif`、`caseof/case/endcase`、`loop/endloop`、`command/endcommand` 这类结构的缩进更稳定。
- 补上了 FISH 代码环境下的注释切换能力，统一使用 `;` 作为行注释前缀。

### Why

满足在文档中插入和阅读 FISH 代码的需求。
