## 5 What
1. 新增 QTMStateToolButton 工具类，支持通过 CSS qproperty-iconNormal/iconChecked/iconHovered 自动切换图标
2. 新增 thinking-white.svg 白色图标资源
3. 将思考按钮从 QToolButton 改为 QTMStateToolButton，图标路径由 CSS 按主题配置
4. 调整 send.svg viewBox 消除图标与按钮之间的空白间隙

## 6 Why
思考按钮的图标切换逻辑硬编码在 C++ 中，不支持按主题配置不同图标。通过 Q_PROPERTY 模式将图标路径下沉到 CSS，实现主题级别的图标配置。

## 7 How
- 新增 QTMStateToolButton（header-only），参照 WindowButton 的 Q_PROPERTY 模式，注册 iconNormal/iconChecked/iconHovered 属性，在 checkStateSet/event 中自动切换图标
- CSS 中使用 qproperty-iconNormal 等设置图标路径，liii 用深色图标+白色选中图标，liii-night 反之
- C++ 中移除硬编码的 setIcon 调用，由 CSS 驱动图标
- send.svg viewBox 从 "0 0 48 48" 调整为 "3 3 42 42" 裁掉空白边距
